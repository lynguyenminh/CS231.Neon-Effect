## Neon effect with Computer Vision
Using Alpha plending to create neon effect.

## 0. My environment
```
OS: Ubuntu20.04
Python: python3.8
```

## 1. How to run this code?
### 1.1. Clone source code and prepare environment

Open terminal and run this script: 
```
git clone https://github.com/lynguyenminh/CS231.Neon-Effect.git && cd CS231.Neon-Effect
pip install -r requirements.txt
```

### 1.2. Run code
Then run the following command to starting code: 
```
streamlit run app.py
```
